# Aiven PostgreSQL Terraform module

Terraform module which creates Aiven PostgreSQL resources

## Usage

```hcl
module "postgresql" {
  source       = "tf-cloud-modules/postgresql/aiven"
  project      = "test"
  service_name = "test"
}
```
